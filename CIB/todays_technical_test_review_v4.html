For a flow such as frontend → backend:

  • The protected destination is the backend.被保护的目标是 backend。
  • The traffic direction is ingress because traffic enters the backend.因为流量进入 backend，所以使用 ingress。

C1. Smallest Positive Interval

Difficulty 5/10

Question

Implement find_smallest_interval(numbers). Return the smallest positive difference between any two values in the list.

Example Input

[1, 6, 4, 8, 2]

Example Output

1

Reasoning

After sorting, the closest pair must be adjacent. Therefore, sort once and compare neighboring values.

Python 3 Solution

from typing import List

def find_smallest_interval(numbers: List[int]) -> int:
    numbers.sort()
    best = float("inf")

    for i in range(1, len(numbers)):
        diff = numbers[i] - numbers[i - 1]
        if 0 < diff < best:
            best = diff

    return int(best)

Complexity

Time: O(n log n). The scan itself is O(n).

C2. Search a Value in Four Zones

Difficulty 5/10

Question

Implement search_value(numbers, value). Return four flags:

  1. 1 if the value appears at an even index.
  2. 1 if the value appears at an odd index.
  3. 1 if the value appears in the first half.
  4. 1 if the value appears in the second half.

All occurrences must be considered.

Example Input

numbers = [5, 8, 3, 8]
value = 8

Example Output

[0, 1, 1, 1]

Reasoning

The value appears at odd indices 1 and 3. Index 1 is in the first half, and index 3 is in the second half.

Python 3 Solution

from typing import List

def search_value(numbers: List[int], value: int) -> List[int]:
    result = [0, 0, 0, 0]
    half = len(numbers) // 2

    for index, number in enumerate(numbers):
        if number != value:
            continue

        if index % 2 == 0:
            result[0] = 1
        else:
            result[1] = 1

        if index < half:
            result[2] = 1
        else:
            result[3] = 1

    return result

Complexity

Time: O(n). Extra space: O(1).

C3. Compute Blood Gene Combinations

Difficulty 7/10

Question

Given the blood types of two parents and one child, return every possible pair of parental genotypes that can produce the child’s phenotype.

Possible phenotype-to-genotype mappings:

A  → AA, AO
B  → BB, BO
AB → AB
O  → OO

A child inherits one gene from each parent. Return the parent genotype pairs sorted first by parent 1 genotype and then by parent 2 genotype. If no valid pair exists, return [["--", "--"]].

Example 1 Input

parent1 = "AB"
parent2 = "A"
child = "A"

Example 1 Output

[["AB", "AA"], ["AB", "AO"]]

Example 2 Input

parent1 = "O"
parent2 = "AB"
child = "O"

Example 2 Output

[["--", "--"]]

Reasoning

  1. Convert each parent phenotype into all possible genotypes.
  2. Enumerate every parent genotype pair.
  3. Simulate one inherited gene from each parent.
  4. Convert the child genes back into a phenotype.
  5. Keep the parent pair if at least one child gene combination matches the required phenotype.

Python 3 Solution

from typing import List

def compute_blood_genes(
    parent1: str,
    parent2: str,
    child: str
) -> List[List[str]]:

    possible = {
        "A": ["AA", "AO"],
        "B": ["BB", "BO"],
        "AB": ["AB"],
        "O": ["OO"],
    }

    def phenotype(gene1: str, gene2: str) -> str:
        genes = {gene1, gene2}

        if genes == {"A", "B"}:
            return "AB"
        if "A" in genes:
            return "A"
        if "B" in genes:
            return "B"
        return "O"

    result = []

    for genes1 in possible[parent1]:
        for genes2 in possible[parent2]:
            valid = any(
                phenotype(gene1, gene2) == child
                for gene1 in genes1
                for gene2 in genes2
            )

            if valid:
                result.append([genes1, genes2])

    result.sort(key=lambda pair: (pair[0], pair[1]))
    return result if result else [["--", "--"]]

Complexity

The state space is constant and very small, so practical runtime is constant. The important part is correct modeling rather than performance.
